File Compressor

This is a File compressor which uses Huffman algorithm for the process of encoding and decoding

You can use this compressor to compress a text file and decomress the same text file

Encode button in the sidebar will take you to the compressor page where you just need to choose the file from your system
and the file will be compressed and will be downloaded to your system

Decode button in the sidebar will take you to the decoder page where you just need to choose the file from your system
and the file will be decompressed and will be downloaded to your system

While decoding and encoding any text file the website will be using HUFFMAN algorithm so the huffman tree will also
be visible to you for that particular text file
